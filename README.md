What is this project about
=========
This is just a project of ChatGPT3.5 demos. 


How to build
============
1. install python 3.10+


2. update pip
```shell
    python -m pip install --upgrade pip
```

3. install packages
```shell
    python -m venv venv
    . venv/bin/activate
    pip install -r requirements.txt
    python test01.py
```

4. run
```shell
    python -m venv venv
    python test01.py
```
